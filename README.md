# Roam — AI-Powered Heritage Tour Guide

Roam is a mobile app that turns city walks into immersive, AI-narrated heritage tours. It combines real-time location tracking, landmark verification via computer vision, and a live conversational AI guide powered by **Gemini 2.5 Live** and **LiveKit**.

## Features

- **Guided Walking Tours** — GPS-based navigation with interactive map, distance tracking, and stop-by-stop progression
- **AI Narration** — Personalized storytelling generated by Gemini, with word-synced audio playback (Charon voice)
- **Landmark Verification** — Point your camera at a landmark; Gemini's vision model confirms you've arrived
- **Live AI Guide** — Real-time voice conversation with an AI tour guide via WebRTC (LiveKit + Gemini 2.5 Live native audio)
- **Personalization** — Choose narration style (Historical, Funny, Poetic, Adventurous) and language (English, Hindi, Marathi, Gujarati)
- **Tour Progress** — Track completed stops, view narration history, and earn completion badges

## Architecture

```
roam/          React Native / Expo mobile app (TypeScript)
agent/         Python LiveKit agent (Gemini 2.5 Flash native audio)
```

### Mobile App (`roam/`)

- **Framework**: Expo 55 + React Native 0.83 + Expo Router (file-based navigation)
- **Maps**: React Native Maps with live user tracking
- **Camera**: expo-camera for landmark capture
- **Audio**: expo-audio for narration playback with word-level sync
- **Voice**: LiveKit React Native SDK for WebRTC
- **State**: Zustand (user preferences, tour progress)
- **UI**: Glassmorphic design with Poppins typography

### Agent (`agent/`)

- **Runtime**: LiveKit Agents framework (Python)
- **Model**: Gemini 2.5 Flash Native Audio Preview for real-time conversation
- **Features**: Context-aware responses based on user preferences, current landmark, and tour progress

## Getting Started

### Prerequisites

- Node.js 18+
- Python 3.10+
- Expo CLI (`npm install -g expo-cli`)
- API keys: Google Gemini, LiveKit

### Mobile App

```bash
cd roam
npm install
npx expo start
```

### Agent

```bash
cd agent
pip install -e .
cp .env.example .env   # add your API keys
python -m roam_guide_agent
```

### Environment Variables

| Variable | Description |
|----------|-------------|
| `GOOGLE_API_KEY` | Google Gemini API key |
| `LIVEKIT_URL` | LiveKit server URL |
| `LIVEKIT_API_KEY` | LiveKit API key |
| `LIVEKIT_API_SECRET` | LiveKit API secret |
